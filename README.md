# WRAD ARMS
### Low Poly FPS Viewmodel Arms | CC0 License

![License: CC0](https://img.shields.io/badge/License-CC0-green.svg)
![Format: GLB/FBX/OBJ](https://img.shields.io/badge/Formats-GLB%20%7C%20FBX%20%7C%20OBJ-blue)

**WRAD ARMS** are free, CC0-licensed low poly FPS viewmodel arms inspired by the **Half-Life 1 / GoldSource** aesthetic. These arms are fully rigged and ready for immediate implementation in **Godot, Unity, Unreal Engine, and Blender**.

Completely free, forever. No attribution required.

---

### FEATURING...
* **FULLY RIGGED IK ARMATURE:** Ready for custom animations and poses.
* **2 SKIN COLORS:** Included variants for project flexibility.
* **1200 TRIANGLES:** Ultra-optimized for modern and retro-style engines.
* **512x512 TEXTURE SIZE:** Maintains that authentic, crunchy 90s feel.
* **MULTIPLE FORMATS:** Comes in **GLB**, **FBX**, and **OBJ**.

### PERFECT FOR...
* **Retro FPS / Boomer Shooters**
* **PS1 / PSX Style Projects**
* **Low Poly Horror & Prototyping**
